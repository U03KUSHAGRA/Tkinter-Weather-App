
# Project Title

Uses the Tkinter library for fetching and displaying real-time weather ata, integrating with the OpenWeatherMap API for a user-specified city.


## Features

- User-Friendly Interface
- Real-Time Data Integration
- API Integration
- Data Handling
- Timezone Adjustment
- Error Handling
